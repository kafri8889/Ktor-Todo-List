## How to use
- Clone repository
- Run `Application.kt`
- Accessing API: `http://0.0.0.0:80/`
- Accessing other route: `http://127.0.0.1/`

API documentation at: `http://127.0.0.1/swagger`

### Database
- H2

### Table
- User
- Todo
- SubTodo
- Category
